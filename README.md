# Projeto_e-comerce

O projeto tem como objetivo final a criação de um e-comerce
Foi dividido em várias partes sendo essas:
1 - Criação da página principal
2- Criação da página de cadastro
3- Criação da página de login
4- Criação das demais páginas auxiliares
5- Elaboração do back-end
7- Implatação do back-end
